#### 介绍
IDE:VScode+platformio<br>
框架：Arduino<br>
芯片：ESP32C3<br>
GUI：LVGL9开源GUI<br>
屏幕：[JDI Memory LCD 低功耗](https://item.taobao.com/item.htm?_u=7mddnvb47eb&id=717207551128&spm=a1z09.2.0.0.3ff52e8dBRWNlM)<br>
交互方式：3向拨轮，未完成<br>

VScode打开，安装插件platformio，然后重启会自动安装esp32核心框架
<br><br>
使用了SPI DMA加速传输<br>
可以选择是否开启抖动<br>
